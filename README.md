# AI Recruitment Frontend

Modern React frontend with Material-UI and voice recognition capabilities.

## Features

- Material-UI v5 components
- SCSS styling
- Voice recognition
- TypeScript
- Vite for fast development
- Responsive design

## Setup

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Environment

The frontend connects to the backend at `http://localhost:5000` by default.

## Technologies

- React 19
- Material-UI v5
- SCSS/Sass
- TypeScript
- Vite
- Emotion (CSS-in-JS)
